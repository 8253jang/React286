Spring 프로젝트 - back_Security_JWT
     1) Spring프로젝트를 인텔리제이에서 오픈한다.
     2)  application.properties를 파일을 확인해서
         ORM설정과 port를 설정한다.
         각자의 프로젝트 상황에 맞게 설정한후 프로젝트를 시작 시킨다.


React프로젝트 - client-jwt-cra
     
   1) React프로젝트를 vscode로 연다
   2) 터미널창을 열고 프로젝트 폴더로 이동
   3) npm i
 
   4)프로젝트 실행 명령어
       npm start

   5) 브라우져에
      http://localhost:3000 


  회원가입 한 후 로그인, 로그아웃 진행
     : 브라우져  개발자도구에서 localStorage 에 저장된 정보를 확인해 본다.

  게시판에 새글 등록, 수정, 삭제 진행해본다.
